# node-http-methods

> Get the http request methods allowed by node http module, which is also used by express etc frameworks

## Install
```
$ npm install --save node-http-methods
```

## Usage
```js
var httpMethods = require('node-http-methods')

console.log(httpMethods);
//=> ['post', 'get', 'put'.....]
```
